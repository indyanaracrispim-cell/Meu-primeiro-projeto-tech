# Enf.Dev Portfolio | Indyanara Crispim

## 1. Objetivo do Projeto
Este projeto consiste no desenvolvimento de uma landing page funcional destinada à apresentação do meu portfólio pessoal. O objetivo central é consolidar as competências técnicas adquiridas durante o curso SCTEC, integrando-as à minha trajetória acadêmica e profissional prévia na área da saúde. Como acadêmica de Enfermagem, busco através deste espaço digital demonstrar como o rigor metodológico, a ética no tratamento de dados e a resiliência do ambiente hospitalar podem ser aplicados de forma estratégica na engenharia de software e na análise de dados.

## 2. Fundamentação Técnica e Metodologia
A arquitetura do projeto foi estruturada utilizando as tecnologias fundamentais da web, garantindo o cumprimento integral dos requisitos de codificação e linguagens solicitados.
* **HTML Semântico: A estrutura foi construída utilizando tags semânticas (header, nav, main, section, article), o que otimiza a acessibilidade e o SEO da página, além de facilitar a manutenção futura do código.
* **CSS e Design Responsivo:  O layout foi desenvolvido com o auxílio de variáveis CSS para padronização de cores e Flexbox para garantir que a interface seja adaptável a diferentes tamanhos de tela.
* **JavaScript: Implementado para garantir interatividade, manipulação do DOM e comportamentos dinâmicos que enriquecem a experiência do usuário.
                 Evitando o uso de div como explicado nas palestras.

## 3. Descrição Detalhada dos Projetos
O portfólio apresenta uma lista com 2 projetos simples que demonstram a versatilidade do meu aprendizado técnico. (Embora construídos em tão pouco tempo e muita dedicação, sendo primeira vez, ao menos consegui entregar algo)
### 3.1. Case: Arquitetura de Dados em SQL
Este projeto simula um sistema de gestão de prontuários hospitalares. Foram desenvolvidas consultas (queries) complexas para a filtragem de pacientes em estado crítico, priorização de atendimento e organização de bancos de dados relacionais. Demonstra domínio em lógica de programação e estruturação de dados. (Fictício). 
### 3.2. Sistema de Monitoramento de Sinais Vitais - UTI Neonatal
Desenvolvido em Python com interface integrada, este sistema automatiza a detecção de desvios hemodinâmicos em recém-nascidos. Através de condicionais lógicas, o software identifica estados de taquicardia ou bradicardia em tempo real, fornecendo alertas visuais imediatos. É um projeto que une diretamente o conhecimento clínico à resolução de problemas via código. (Ideia)

Este portfólio é o marco inicial da busca pela minha futura carreira voltada para a criação de soluções tecnológicas que impactem positivamente a sociedade e as organizações.
